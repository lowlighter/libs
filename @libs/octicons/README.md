# ㊗️ Octicons

[![JSR](https://jsr.io/badges/@libs/octicons)](https://jsr.io/@libs/octicons) [![JSR Score](https://jsr.io/badges/@libs/octicons/score)](https://jsr.io/@libs/octicons)

GitHub's [Octicons](https://primer.style/octicons) repackaged as individual SVG rendering functions.

- [`📚 Documentation`](https://jsr.io/@libs/octicons/doc)

## ✨ Usage

```ts
import { markGithub } from "@libs/octicons"
console.log(markGithub({ ariaLabel: "GitHub" }))
// <svg version="1.1" width="16" height="16" ...>...</svg>
```
